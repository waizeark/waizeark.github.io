サイトマップ

開発者向けの無料オンラインツールを提供しています。 JSON整形、コード生成、エンコード変換などをブラウザ上で安全に利用できます。

主要ツール

  • トップページ
    サイト概要と各種ツールへの入り口
  • JSON Formatter
    JSONの整形・検証をブラウザ上で実行可能
  • C# JSON Tool
    JSONからC#クラス生成をサポート
  • TypeScript JSON Tool
    JSONからTypeScript型定義を生成
  • Base64 Encoder/Decoder
    Base64変換ツール
  • UUID Generator
    UUIDを即時生成可能
